#+TITLE: Exercise Tracker
#+STARTUP: indent

A little hobby project because I'm tired of using my notes app to track my exercises in the gym.
Still very much a work in progress; right now I'm just exploring a few ideas, what it could look like, what's possible, how it should be structured, etc.

* Requirements
- node ^20.19.0 || >=22.12.0 (using version 26.3.0)
- npm
- android studio

* Project Setup
#+begin_src sh
  npm install
#+end_src

* Dev (browser)
#+begin_src sh
  npm run dev
#+end_src

* Build
** browser
#+begin_src sh
  npm run build
#+end_src

** android
#+begin_src sh
  # builds the html/css/js part of the app
  npm run build

  # copies web app build into native app
  # and updates capacitor plugins + package.json dependencies
  npm run cap-sync

  # opens android studio with the project loaded
  # right now my development workflow is:
  #   connect my phone via usb,
  #   select my phone in devices,
  #   hit run in android studio
  npm run cap-open-android
#+end_src
